# Yemeksepeti Web Sitesi Tasarımı

Bu proje, Web Tasarım dersi final ödevi olarak hazırlanmıştır.  
Projede yemek siparişi konseptine uygun bir web sitesi tasarımı geliştirilmiştir.

## Kullanılan Teknolojiler
- HTML
- CSS
- JavaScript

## Amaç
Bu projede amaç, ders boyunca öğrenilen web tasarım konularını kullanarak görsel açıdan düzenli ve kullanıcı dostu bir site oluşturmaktır.

## İçerik
- Ana sayfa tasarımı
- Yemek ve restoran görselleri
- CSS ile sayfa düzeni ve stil tasarımı
- JavaScript ile temel etkileşimler

## Çalıştırma
Projeyi çalıştırmak için `index.html` dosyasını tarayıcıda açmanız yeterlidir.

## Not
Bu proje eğitim amacıyla hazırlanmıştır.
